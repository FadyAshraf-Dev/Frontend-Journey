---

## ⚙️ How to View Locally

To explore any template or task on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate into the folder:**
   ```bash
   cd Frontend-Journey
   ```
3. **Open files:**
   Simply open any `index.html` file within the folders directly in your browser or run them using the VS Code **Live Server** extension.
